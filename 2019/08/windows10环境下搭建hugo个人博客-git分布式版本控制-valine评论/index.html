windows10环境下搭建Hugo个人博客+git分布式版本控制+valine评论

八月 23, 2019 发布在 个人博客

自己在windows平台上搭建hugo个人博客的教程，这个博客部署在远端github的仓库上，添加了Valine评论和页面内音乐。详细记录下搭建博客的大概流程，以及搭建过程中遇到的困难，给想搭建自己博客的童鞋们做个铺路。

可以看下B站up主CodeSheep的视频和saquarius的教程，解决了我很多的迷惑，一路跟着搭建的，非常感谢！

目录

  • 准备工作

准备工作

  • 下载hugo安装包
  • 创建Github账户
  • 创建LeanCloud账户（用于个人评论系统）
  • 下载markdown ，下载Notepad++加markdown插件就可以了

个人博客的搭建

安装git

  • 安装git，具体流程可参考廖雪峰的Git教程
  • 学习git基本的语法,可以稍后学习。
  • 添加git到系统环境变量PATH中，添加方法：桌面我的电脑，右键属性，左边高级系统设置，右下角环境变量。Path编辑，浏览确定。需要添加的文件夹如图所示：

安装hugo

  • 在hugo下载地址下载Hugo安装包，选择Windows64版本的程序包。
  • 解压到文件夹中,路径不要有中文，后面会用到这个路径，记得以后不要删除这个文件夹和文件，
  • 添加hugo到系统环境变量PATH中，添加方法：桌面我的电脑，右键属性，左边高级系统设置，右下角环境变量。Path编辑，浏览确定。

  • 命令控制台输入：hugo version出现以下提示则hugo安装成功。

  • 安装成功后，用Hugo来创建自己的博客，命令如下图所示：hugo new site myblog,myblog为自己私人博客的文件夹名称，在之前可以选择合适的地址来放博客。如图：

  • 在myblog里面，可以看到如下内容：

  • 设置主题，去https://themes.gohugo.io/选择自己喜欢的主题,这里以tranquilpeak主题为例。按照指引操作，如图所示：

    cd themes
git clone https://github.com/kakawait/hugo-tranquilpeak-theme.git

  • 打开myblog文件夹下的themes主题文件夹，改名为：tranquilpeak。

  • 下面是创建文章，首先是创建文章的模板，在\myblog\themes\tranquilpeak\archetypes\post.md,我的模板设置如下：

    ---
title: "{{ replace .TranslationBaseName "-" " " | title }}"
date: {{ .Date }}
categories:
- category
- subcategory
tags:
- tag1
- tag2
keywords:
- tech
# 每次编辑文档时请删去这些解释的注释
# 缩略图，最好采用宽度为750的图片，可以使用本地文件或者使用git图床
# 使用本地图库，请把图片存在themes/static/images文件夹下，并且用下面的路径引用
thumbnailImage: /images/cover.jpg
# 缩略图位置，我觉得放在上面好看，如果用的小图片作缩略图
# 可以放在（left或者right），设置此项会取代配置文件中默认在下边的设置
thumbnailImagePosition: top
# 这个下边默认就好了，暂时不清楚有什么影响
autoThumbnailImage: yes
metaAlignment: center
# 封面图片，如果选用本地文件的话，和缩略图同样的路径，21：9的图片效果很好看
coverImage: /images/cover.jpg
# 封面图片的描述引号中的文字可以自己选
coverCaption: "文章cover图片描述"
# in表示文章标题显示在封面图片上，out表示显示在封面图片的外边，也就是文章开头
coverMeta: in
# 封面图片的覆盖大小，直接默认为full吧，21:9的图片这样设置显示效果真的不错
coverSize: full
---

  • 安装插件MarkdownViewer++，在菜单栏选择插件 → 插件管理，找到MarkdownViewer++，右上角点击安装，完成之后重启Notepad++即可在工具栏找到预览Markdown的功能。
